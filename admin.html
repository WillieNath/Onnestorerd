🔐 Credenciales del admin


Las credenciales se guardan en localStorage del navegador. Configura una contraseña segura y guárdala también en un lugar seguro.

📋 Información técnica



  
📦 Config: localStorage → clave "siteConfig"

  
🗄️ Archivos grandes: IndexedDB → base "OnneStoreRD"

  
🏛️ Admin DB: IndexedDB → base "storeAdminDB" (15 tablas)

  
🔑 Clave credenciales: localStorage.adminCreds

  
🛒 Carrito: localStorage.cartState (por cliente)
